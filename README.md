# 💪 Solo Level-Up Fitness Tracker

A web-based fitness tracking application inspired by *Solo Leveling* that helps users set workout goals, track daily progress, and stay motivated through a gamified experience.

## 🚀 Features

* User Login & Registration
* Add Custom Fitness Goals
* Track Daily Workout Progress
* Progress Bars & Level-Up Effects
* Daily Progress History
* Responsive Anime-Inspired UI
* LocalStorage Data Saving

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* LocalStorage

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in your browser

## 🎯 Purpose

The project is designed to make fitness tracking more interactive and engaging using a gamified system inspired by anime-style progression.
